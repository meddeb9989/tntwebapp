About The Project

Say Goodbye to the Restaurant Tickets it's time to deal with RFID Cards.

TAN & TECH Team this is our web application for Aventix project INSA LYON MS IF 2017.

This web app is for the clients of Aventix (Traders, Employer, Employee). And the idea about this project is to change the restaurant tickets, that Employer give to their Employees, by an RFID Cards which can be charged or opposed via this website, also Employees can see their balance and the Traders can see their transactions .
